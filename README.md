# Data Analyst Portfolio

This repository showcases my ongoing learning and practice as an aspiring Data Analyst.
It includes structured SQL solutions and pandas problem-based notebooks focused on
developing strong data manipulation and querying skills.

The emphasis is on writing clean, readable queries, applying pandas for data
transformations, and building consistency through regular practice.

---

## Contents

### SQL
- Topics include:
  - SELECT, WHERE, ORDER BY
  - JOINs
  - GROUP BY & HAVING
  - Subqueries
  - Window functions

Folder: `sql/sql50/`
- Question-wise SQL solutions based on LeetCode SQL50
